Alojamiento Web

Para crear un dominio Web bajo uni.edu.ni, es necesario solicitarlo por escrito al Secretario General Ing. Diego Muñoz y al Director DTIC Ing. Sixto Chavarría, un email se considera como una solicitud escrita, con los siguientes datos:

  • Explicación de cuales son los motivos y finalidad del dominio Web.
  • Nombre del dominio ejemplo: www.tusitio.uni.edu.ni.
  • Responsable para la administración del dominio Web (Webmaster).

    Las facilidades que nosotros ofrecemos son las siguientes:

  • Alojamiento de sitio Web en Nuestro Web Server.
  • Creación de una cuenta de correo Institucional para el encargado de administrar el sitio Web.
  • Proveer el software necesario para la administración del sitio Web (Instalador del Webdrive, versión 5.32).
